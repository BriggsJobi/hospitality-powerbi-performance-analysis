PROJECT TITLE
Urbanstay Hospitality Performance & Competitor Analysis Dashboard (Power BI)

PROJECT SUMMARY
This project analyzes hotel performance, customer behavior, booking channels, and competitor pricing strategies using Power BI.
The goal is to identify revenue trends, customer segments, operational gaps, and competitive positioning in order to support data-driven business decisions in the hospitality industry.
The dashboard provides insights across three main areas: overall business performance, customer behavior patterns, and competitor benchmarking.

PROJECT OBJECTIVES
▪ Analyze overall revenue, profit, guest volume, and revenue loss trends.
▪ Identify dominant customer demographics, nationalities, and booking behaviors.
▪ Evaluate performance across booking channels and room types.
▪ Assess competitor pricing strategies, room rates, and booking volumes.
▪ Translate analytical findings into actionable business recommendations.

TOOLS AND TECHNOLOGIES USED
▪ Power BI — data modeling, DAX measures, and interactive dashboards
▪ Power Query — data cleaning and transformation
▪ DAX — KPI calculations and performance metrics
▪ Excel / CSV — source data format

DATA DESCRIPTION
The dataset represents multi-year hotel booking and competitor performance data and includes:
▪ Booking details (dates, channels, cancellations, stay duration)
▪ Guest demographics (age group, nationality, loyalty status)
▪ Room types and travel purposes
▪ Revenue, profit, and revenue loss metrics
▪ Competitor hotel pricing strategies, ratings, and room rates

The data was cleaned, transformed, and modeled into fact and dimension tables to support accurate trend analysis and cross-filtering across dashboard pages.

KEY DELIVERABLES
▪ Executive Overview Dashboard
▪ Customer Analysis Dashboard
▪ Competitor Analysis Dashboard
▪ Executive Report Page with Insights and Strategic Recommendations
